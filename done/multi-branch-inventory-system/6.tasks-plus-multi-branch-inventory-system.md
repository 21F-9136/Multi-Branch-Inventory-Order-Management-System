## Error Handling Standard

All API responses must follow a unified error structure.

Example:


{
"code": "INSUFFICIENT_STOCK",
"message": "Requested quantity exceeds available stock",
"request_id": "abc123"
}


Common error codes:

- VALIDATION_ERROR
- PERMISSION_DENIED
- RESOURCE_NOT_FOUND
- INSUFFICIENT_STOCK
- SYSTEM_ERROR
